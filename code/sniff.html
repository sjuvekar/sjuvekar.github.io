Sniff: Eclipse plugin for Code Search in Java

Sniff is an Eclipse plugin to search small code sniffets in a Java project using google-like queries. Sniff helps a coder to

  • Type an english language query to search specific code usage in an API. e.g. "Parse AST in JDOM" or "open directory dialog"
  • The plugin searches through its existing codebase using NLP to find a small code snippets most likely performing the required task
  • The plugin also uses pattern matching to ranks these snippets. The user can examine each snippet in the result before including that in his/her own code
